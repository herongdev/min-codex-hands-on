# I18N Glossary

Chinese is the source language for the course. Use this glossary when translating course pages, blog entries, and product pages so readers see the same terms across all surfaces.

| 中文 | English | Notes |
| ---- | ------- | ----- |
| 智能体编程 | agent programming | Use for the general ability to build tool-using agents. |
| 代码智能体 | Coding Agent | Capitalize when referring to the product category. |
| Mini Codex | Mini Codex | Keep as product/project name. |
| 任务运行 | Agent Run | Use when describing one trackable task execution. |
| 状态机 | state machine | Use lowercase in prose. |
| 工具注册 | Tool Registry | Use when naming the component. |
| 权限审批 | permission gate | Use for approval boundaries. |
| 运行时策略 | runtime policy | Distinguish from prompt instructions. |
| 改动草稿 | change draft | Use before introducing `diff`; use "diff review" for review flow. |
| 改动审查 | diff review | Use for human review of proposed changes. |
| 验证 | verify / verification | Use "verify" for the action, "verification" for the capability. |
| 过程记录 | trace | Use for replayable execution records. |
| 复盘评测 | eval | Use when tied to scoring or regression evidence. |
| 练习案例 | demo case | Use for runnable practice tasks. |
| 复盘台账 | eval ledger | Use for the table of evidence and scores. |
| 桌面工作台 | desktop workbench | Use for the UI that observes tasks and results. |
| 进阶课 | advanced course | Do not expose private paths or unpublished chapter lists. |

## Translation Rules

- Keep Chinese as the source of truth until a full translation workflow exists.
- Translate page titles, descriptions, CTA text, and summaries first.
- Translate labs one by one only after the course reader is stable.
- Do not publish private advanced-course paths in any language.
- Avoid job guarantees. Say the course provides portfolio/interview evidence, not guaranteed offers.
