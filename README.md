# Mini Codex v2 基础课

欢迎来到 Mini Codex 主线课程。

这套课会带你从第一个目录开始，逐步做出一个能跑、可审批、可验证、可复盘的 Mini Codex。

目标：[做出一个能跑的 Mini Codex](./00-read-me-first.md)。第一步：进入 Lab 00 创建目录。

## 课程顺序

```text
00 起步说明
01-10 基础闭环：保存任务 -> 等你确认 -> 修改代码 -> 运行检查 -> 做成界面
11 对照真实 AI 编程工具，判断还能升级什么
12-13 用练习案例复盘效果
```

## 你会做出什么

完成基础课后，你会有一个能跑的 Mini Codex：

```text
输入一句代码修改需求
-> 保存任务
-> 给出计划
-> 等你确认
-> 修改文件
-> 运行检查
-> 保存过程
-> 桌面界面查看
```

里面有价值的判断已经并入 v2 基础课；进阶课独立维护，暂不随本仓库开源。
