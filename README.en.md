# Mini Codex v2 Foundation Course

[中文版](./README.md)

Welcome to the Mini Codex foundation course.

This course guides you from an empty folder to a small local coding agent that can plan, ask for approval, modify code, run checks, and keep a replayable process record.

Goal: [Build a Working Mini Codex](./00-read-me-first.en.md). The first step is Lab 00, where you create the practice workspace.

## Course Order

```text
00 orientation
01-10 foundation loop: save task -> wait for approval -> modify code -> run checks -> build a workbench
11 compare the boundary with real Codex-style tools
12-13 validate the result with demo cases and an eval ledger
```

## What You Will Build

By the end of the foundation course, you will have a working Mini Codex:

```text
enter one coding request
-> save it as a task
-> generate a plan
-> wait for your approval
-> modify files
-> run checks
-> save the process
-> inspect it in a desktop workbench
```

The advanced course is maintained separately and is not included in this open-source repository.
