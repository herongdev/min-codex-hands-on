# 开源发布说明

本仓库开源 Mini Codex v2 基础课，目标是帮助读者从零做出一个能跑、可审批、可验证、可复盘的 Mini Codex。

## 开源范围

包含：

- 基础课 README 和起步说明
- Lab 00 到 Lab 08
- 真实 Codex 功能升级包
- 练习案例和复盘台账模板

不包含：

- v2 Advanced 进阶课正文
- 私有课程路线、内部发布材料和未公开商业课内容

## 发布建议

推荐展示结构：

1. GitHub 仓库作为开源内容源。
2. `mianshiti` 展示课程入口和适合人群。
3. `blog` 发布课程介绍文章并链接 GitHub。

这样可以让 GitHub 负责版本维护，两个站点负责转化和说明，避免多处复制后内容漂移。
